Quick Start

For detect the edges in a 2D image follow this steps:

  1. Select a 2D input image on Param. tab.
  2. Select the threshold value in the same tab (also you can select if you are detecting a first order edge).
  3. Select the method to apply and if you want to draw the normals of the edges. Based on the method, you can set some parameters.
  4. Press the run button.
