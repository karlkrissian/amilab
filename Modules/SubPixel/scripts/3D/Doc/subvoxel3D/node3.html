Param. tab

In the Param. tab you can select an input image for the subvoxel edges detection. Is necessary to set a threshold value for the detection (25 by default).

After to select these parameters, you can choose the method to apply, to check if you want to draw the volume, the edges or the normals of the edges.

In the visualization box panel you can check draw:

  • Volume.
  • Edges.
  • Normals.

At the bottom of this tab is the run button. Also, you can find in the right of the button run, the button reload. You can use it if you change any parameter of the Set. tab.
