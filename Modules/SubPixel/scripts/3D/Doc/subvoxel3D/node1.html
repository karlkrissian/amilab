Quick Start

For detect the edges in a 3D image follow this steps:

  1. Select a 3D input image on Param. tab.
  2. Select the threshold value in the same tab.
  3. Select the method to apply and if you want to draw the the volume, edges or normals.
  4. Press the run button.
