Param page

  1. Set the seed point with the Set Seed Point button. An image appear, select with the central mouse button the seed point and then press the same button once more.
  2. Select the stopping time value. Fast Marching algorithm is terminated when the value of the smallest trial point is greater than the stopping value.
  3. Select the lower and the upper threshold for Binary Image Filter. It's used to binarize the output image of the FastMarching by thresholding
  4. Select the seed to set the container of Trial Points representing the initial front.
  5. Run the filter with Run button.
  6. Display the result with Display button.
