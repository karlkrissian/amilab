Parameters

The parameters are:

  • input image
  • Gaussian standard deviation
  • Order of derivation in X
  • Order of derivation in Y
  • Order of derivation in Z
  • Auto: automatic computation after changing a parameter
  • Auto drag: automatic computation while changing a parameter with a scale
  • Apply
