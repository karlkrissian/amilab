Input Intensity-based Speed

If you want to use speed image, you must mark ``Use speed''. Then, select the minimum and the maximum intensity that are used. Select the input speed value.

Select the step size for the evolution. Select the maxlength, maximal Euclidean distance of the path.

Run with the Run button. Display the result with the Display button.
