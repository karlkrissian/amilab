Quick Start

The different steps for obtaining the Recursive Gaussian Filter are:

  1. Select and load the input image.
  2. Select the dimension of the image (2D or 3D).
  3. Select the sigma value, measured in world coordinates, of the Gaussian kernel. The default is 1.0.
  4. Select the flag for normalizing the gaussian over scale space.
  5. Select the Order of the Gaussian to convolve with.
  6. Run the filter.
  7. Display the result.
