Quick Start

The different steps for obtaining the Weighted Local Stats are:

  1. Select and load the input image.
  2. Select and load the wights input image.
  3. Select the dimension of the image (2D or 3D).
  4. Select the prefix for the output image name.
  5. Select the size of the window.
  6. Select the minimal weight.
  7. Run the filter.
  8. Display the result.
