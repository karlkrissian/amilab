Set. tab

In the Set. tab you can change the settings of the edges and the normals. First, you must check that you want change the default settings. In both cases, edges and normals, you can select:
  • Color: You can set the color for the edges or normals in the RGB scale. Also, you can indicate the value of the alpha channel for transparency (0 for transparent and 255 for opaque).
  • Properties: Here you can set the thickness and the style of the line.

When you apply the method you will see the changes established.
