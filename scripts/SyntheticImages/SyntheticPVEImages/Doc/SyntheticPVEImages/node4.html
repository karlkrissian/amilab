Func tab

In the Func tab you can create an analytic function object. There are four functions with some parameters:
  • Line: The normal parameters and y axis cut point.
  • Circle: Center and the radius.
  • Vessel: Center, angle and thickness.
  • Ring: Center, radius and thickness.
  • Cube: Center and x,y,z dimensions.
  • Cylinder: Radius and x,y,z directions.
  • Sphere: Center and radius.
  • Torus: Center, minimum and maximum radius.

All analytic functions have a reload option. If you check it, you can see the changes on the image when modify a parameter.
