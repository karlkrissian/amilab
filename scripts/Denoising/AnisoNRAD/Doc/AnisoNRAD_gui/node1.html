Quick Start

  1. Select the input noisy image in the Param tab;
  2. Select the parameters of the equation;
  3. In Controls tab, reload the initial image, select the Region of Interest for estimating the noise, and apply the filter;
  4. Display the result;
  5. Save it to disk or export it to a global variable.
