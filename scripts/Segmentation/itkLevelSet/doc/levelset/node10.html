LevelSet page

To apply the filter you must to set some parameters:
  1. Select ErrorRMS, the maximum error allowed in the solution.
  2. Select the number of iterations that the filter will run.
  3. Select the scaling of the advection field. Setting the FeatureScaling parameter will override any existing value for AdvectionScaling.
  4. Select the scaling of the curvature. Use this parameter to increase the influence of curvature on the movement of the surface. Higher values relative to Advection and Propagation values will give smoother surfaces.
  5. Select the scaling of the propagation speed. Setting the FeatureScaling parameter overrides any previous values set for PropagationScaling.
Now run the filter with the Run button. You can see the result with the Display button.
