Quick Start

The different steps for obtaining the Fast Marching are:

  1. Select the mode of the load input image (NormGradient or User Velocity Image).
    1. If you select NormGradient follow the SigmoidFilter help to know how to use the SigmoidFilter gui.
    2. If you select User Velocity Image select the image to load and set the dimension of the image (2D or 3D).
  2. Set the seed point with the Set Seed Point button.
  3. Select the stopping time value.
  4. Select the lower and the upper threshold for Binary Image Filter.
  5. Select the seed to set the container of Trial Points representing the initial front.
  6. Run the filter.
  7. Display the result.
