Quick Start

The different steps for obtaining the Wathershed Filter are:

  1. Select and load the input image.
  2. Select the dimension of the image (2D or 3D).
  3. Select the gradient mode (normgradient).
  4. Select the standard deviation for the normgradient.
  5. Select the input thresholding parameter. Units are a percentage of the maximum depth in the image.
  6. Select the flood level for generating the merge tree from the initial segmentation .
  7. Run the filter.
  8. Display the result.
  9. You can save the image result and the gradient result.
