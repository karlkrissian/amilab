Quick Start

The different steps for obtaining the segmentation are:

  1. Select and load the input image,
  2. Select the initial level set either from spheres or from the zero-crossing of an image,
  3. Compute statistics of the intensity statistic within the initial contour,
  4. optionally tune the equation parameters,
  5. evolve the equation,
  6. display the resulting surface.
