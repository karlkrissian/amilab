Quick Start

The different steps for obtaining the Path are:

  1. Select and load the input image.
  2. Select and load the speed image if you want to use it.
  3. Select the start point and the end point.
  4. If you want to save the fastmarching output mark Save FastMarching Output.
  5. Select epsilon value.
  6. Select maxcost value.
  7. If you load the speed image mark Use Speed, then select the minimum and maximum intensity and the input speed value.
  8. Select the step size.
  9. Select the maxlength.
  10. Run the filter.
  11. Display the result.
