Detailed Description

Creates a path starting at a point and following the displacements given by the input vector field image. If the input is scalar the corresponding vector field is computed from its gradient by local linear interpolation. The tracking stops in one of the following cases:
  1. The maximal distance is reached.
  2. The closest voxel to the current location has a negative intensity.
  3. The current gradient is very low.
