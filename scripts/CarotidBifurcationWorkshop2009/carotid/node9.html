Vessel intensity range

You can change the Min and Max intensity range to compute Vesselness. Min = 1150 and Max = 1600 are set by default.

Use the Vesselness button to compute the Vesselness solution.

You can save the result using Save button.

Then you can read the output vesselness solution using the Read button.
