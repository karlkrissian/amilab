Interface tabs

For use this script you need to check the configuration in carotidchallenge_config.amil. You must to specify:
  1. Data_dir = data path folder
  2. Groundtruth_dir = groundtruth path folder (lumen)
  3. Results_dir = results path folfer
  4. Evaluation_bindir = evaluation build path folder
  5. Evaluation_scriptdir = evaluation scripts path folder

The interface is divided into several pages.


Subsections
  • Input page
    • data information

  • Prob+Speed page
    • LocalStats
    • Vesselness
    • Vessel intensity range

  • Run page
    • Paths(Vesselness+Prob)
    • Path
    • Junction
    • LevelSets from paths

  • Adv page
  • Help page
